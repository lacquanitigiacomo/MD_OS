# Release

## Nome

REFACTOR TOTALE V2

## Criteri

La release è accettabile se:

1. la struttura resta stabile;
2. i dataset sono interrogabili;
3. almeno 10 innovazioni incrementali sono tracciate;
4. gli agenti principali sono mappati;
5. esiste un modello memoria;
6. esiste un modello retrieval;
7. esistono blocker release;
8. esiste beta loop;
9. esistono script locali gratuiti;
10. il comando REFACTOR produce output misurabile.
