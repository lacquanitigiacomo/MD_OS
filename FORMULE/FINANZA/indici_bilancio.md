# FORMULE: indici_bilancio

## Regola generale
Applicare le formule solo quando tutti i dati necessari sono presenti e leggibili. Se manca un dato, dichiarare il calcolo non eseguibile.

## Formule principali
- ROI = risultato_operativo / capitale_investito_netto * 100
- ROE = utile_netto / patrimonio_netto * 100
- ROS = risultato_operativo / ricavi_netto * 100
- Current Ratio = attività_correnti / passività_correnti
- Debt/Equity = debiti_totali / patrimonio_netto
- PFN = debiti_finanziari - disponibilità_liquide - attività_finanziarie_liquide

## Output formula
- nome formula;
- dati necessari;
- dati usati;
- risultato;
- limite interpretativo.
