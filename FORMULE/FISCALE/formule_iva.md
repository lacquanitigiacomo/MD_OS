# FORMULE: formule_iva

## Regola generale
Applicare le formule solo quando tutti i dati necessari sono presenti e leggibili. Se manca un dato, dichiarare il calcolo non eseguibile.

## Formule principali
- IVA = imponibile * aliquota_iva
- totale_fattura = imponibile + IVA
- imponibile = totale_fattura / (1 + aliquota_iva)

## Output formula
- nome formula;
- dati necessari;
- dati usati;
- risultato;
- limite interpretativo.
