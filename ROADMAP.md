# Roadmap

## Fase 1 — Stabilizzazione

- congelare struttura;
- mantenere root minimale;
- normalizzare dataset;
- validare file obbligatori.

## Fase 2 — Cognizione

- completare modello memoria;
- completare retrieval interno;
- usare `RELATIONS.csv` come knowledge graph minimo;
- registrare eventi di apprendimento in `LEARNING_EVENTS.jsonl`.

## Fase 3 — Runtime leggero

- query dataset;
- profilo sistema automatico;
- report REFACTOR;
- check release.

## Fase 4 — Beta e release

- scenari beta;
- blocker release;
- readiness report;
- nuova release solo con dati aggiornati.
