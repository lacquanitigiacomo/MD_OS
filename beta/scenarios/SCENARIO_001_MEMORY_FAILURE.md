# Scenario 001 - Memory Failure

## Goal

Test whether MD_OS can detect missing memory and recover using guided learning.

## Simulated Situation

The user asks MD_OS to continue a previous architecture discussion.

The system lacks:

- memory linkage
- retrieval layer
- decision reconstruction

## Expected Agent Activation

- interaction_agent
- memory_agent
- retrieval_agent
- knowledge_gap_agent
- senior_ai_architect
- production_realist

## Expected Behavior

1. detect missing knowledge
2. identify affected areas
3. ask targeted reconstruction questions
4. search existing datasets
5. rebuild missing context
6. update memory datasets

## Failure Conditions

- hallucinated continuity
- pretending memory exists
- no gap detection
- no dataset update

## Required Outputs

- missing knowledge report
- recovery questions
- updated datasets
- memory patch proposal
