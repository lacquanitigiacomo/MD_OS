# FUNZIONE: classifica_non_conformita

## Scopo
Eseguire l'azione operativa `classifica_non_conformita` in modo controllato, tracciabile e riutilizzabile dagli agenti.

## Input
- dati disponibili;
- fonte del dato;
- contesto della richiesta;
- eventuali limiti dichiarati.

## Procedura
1. Verificare se i dati necessari sono presenti.
2. Escludere dati dubbi o non leggibili.
3. Applicare la logica della funzione.
4. Restituire risultato e motivazione.
5. Dichiarare dati mancanti e affidabilità.

## Output
- risultato;
- dati usati;
- dati mancanti;
- affidabilità;
- note operative.
