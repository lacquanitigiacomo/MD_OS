# Orchestratore

## Scopo

Trasformare ogni richiesta in un pacchetto runtime leggero.

## Flusso

```text
RICHIESTA
→ intenzione
→ professioni necessarie
→ profili cognitivi
→ skill operative
→ dataset minimi
→ matrice di scelta
→ risposta
→ apprendimento
```

## Pacchetto Runtime

| Campo | Descrizione |
|---|---|
| intenzione | cosa serve davvero |
| professioni | chi lavora |
| profili | come pensa |
| skill | cosa fa |
| dataset | cosa ricorda |
| matrice | come valuta |
| output | cosa produce |
| apprendimento | cosa resta |
