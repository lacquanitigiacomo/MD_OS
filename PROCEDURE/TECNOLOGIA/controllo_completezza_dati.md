# PROCEDURA: controllo_completezza_dati

## Scopo
Guidare l'agente nella procedura `controllo_completezza_dati`.

## Sequenza operativa
1. Identificare documento, dataset o richiesta.
2. Estrarre i dati essenziali.
3. Verificare completezza e coerenza.
4. Applicare eventuali formule collegate.
5. Segnalare anomalie.
6. Produrre output strutturato.

## Controlli
- completezza;
- coerenza;
- tracciabilità;
- affidabilità.
