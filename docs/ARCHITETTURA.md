# Architettura MD_OS

## Principio

MD_OS usa una struttura stabile e data-first. I dataset sono la fonte primaria. I documenti spiegano regole e protocolli. Gli agenti sono profili derivati dai dati più eventuali override.

## Livelli

| Livello | Funzione |
|---|---|
| root | orientamento rapido |
| datasets | memoria interrogabile |
| docs | protocolli stabili |
| agents | override e specializzazioni |
| engine | strumenti locali |
| beta | test e debug |
| release | criteri e readiness |
