# Modello Retrieval

## Scopo

Recuperare conoscenza interna senza dipendere da servizi a pagamento.

## Fase 1

- CSV query locale;
- relazioni in `RELATIONS.csv`;
- eventi append-only in `LEARNING_EVENTS.jsonl`.

## Fase 2

- indicizzazione semantica locale;
- ranking per rilevanza;
- sintesi multi-dataset.
