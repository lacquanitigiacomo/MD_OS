# Protocollo REFACTOR

## Definizione

`REFACTOR` è una routine operativa. Deve produrre almeno 10 avanzamenti misurabili.

## Output obbligatori

- 10 avanzamenti;
- dataset target;
- pattern rilevati;
- aggiornamenti agenti;
- impatto release;
- prossima patch consigliata.

## Regola

Un refactor senza dati aggiornabili non è valido.
