# Simulazione 10G — Agenti Professionali e Profili

## Scopo
Simulare un sistema avanzato in cui gli agenti sono professioni reali in italiano e i profili sono modalita cognitive applicabili su richiesta.

---

## Regola Base

```text
PROFESSIONE + PROFILO + SKILL + MEMORIA + MATRICE = AGENTE OPERATIVO
```

---

## Risultato della Simulazione

La simulazione funziona perche separa chiaramente:

- mestiere reale;
- profilo cognitivo;
- skill operative;
- memoria attiva;
- matrice di valutazione;
- output atteso.

Questo permette agli agenti professionali di collaborare allo sviluppo chiamati su richiesta, senza trasformare i nomi simbolici in ruoli opachi.

## Regola Finale

Ogni nuova funzione di MD_OS deve dichiarare almeno:

| Campo | Obbligo |
|---|---|
| Professione coinvolta | chi lavora |
| Profilo attivo | come pensa |
| Skill usata | cosa fa |
| Memoria richiamata | cosa sa |
| Matrice applicata | come valuta |
| Output prodotto | cosa consegna |
| Apprendimento salvato | cosa resta |
