# Cervello Operativo — MD_OS 10G

## Scopo
Trasformare richieste, dati e memoria in azioni cognitive organizzate.

---

# Livelli Cognitivi

| Livello | Funzione |
|---|---|
| L1 | capire richiesta |
| L2 | trovare memoria utile |
| L3 | costruire squadra |
| L4 | scegliere matrici |
| L5 | produrre azione verificata |

---

# Sequenza Operativa

```text
INPUT
↓
INTENZIONE
↓
MEMORIA
↓
SQUADRA
↓
MATRICE
↓
AZIONE
↓
VERIFICA
↓
APPRENDIMENTO
```

---

# Matrici Base

| Matrice | Uso |
|---|---|
| impatto / sforzo | priorita |
| rischio / reversibilita | decisioni |
| profondita / velocita | risposta |
| segnale / rumore | ricerca |
| novita / fattibilita | innovazione |

---

# Scheda Operativa

| Campo | Contenuto |
|---|---|
| richiesta | input originale |
| intenzione | bisogno reale |
| memoria attiva | dataset richiamati |
| squadra attiva | agenti coinvolti |
| matrice scelta | metodo di valutazione |
| output | risultato |
| apprendimento | memoria salvabile |
